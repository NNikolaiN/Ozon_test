# Ozon_test

## Требования

- Python 3.10+
- Git

**Проверка:**
```
python --version
git --version
```
## Быстрый старт

### 1. Клонирование репозитория
```bash
git clone https://github.com/NNikolaiN/Ozon_test
cd Ozon_test
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```
### 3. Запуск тестов
```bash
pytest -v
```
